## 模块二：Phase 1 运营策略详解

## 2.1 验证期运营手册（0-3个月）

> **核心目标**：验证需求真实存在，跑通第一个完整交易闭环，积累10-20个种子用户和5-10个核心设计师

### 2.1.1 冷启动策略：从0到第一单

#### Step 1: 种子客户获取（第1-2周）

**渠道1：你的现有资源池**

```
盘点清单：
□ 过往项目的甲方联系人（HRB、YC等项目的开发商）
□ 同行设计师群（微信群、QQ群）
□ 校友资源（建筑院校同学群）
□ 知乎/公众号粉丝（如有）
□ 脉脉/LinkedIn建筑设计人脉

执行动作：
1. 制作「RMO-AI平台介绍」1页PDF
   - 标题：建筑设计任务众包平台 | 您的弹性设计团队
   - 核心价值：快速对接优质设计师，AI提效，质量把控
   - 你的背书：XX年建筑设计经验，操盘XX万㎡项目
   - 联系方式：微信二维码

2. 一对一私聊话术模板：

   「X总/X工，我是XX，最近在做一个新尝试。
   
   简单说，这是一个建筑设计任务的"滴滴"平台——
   您有做不过来的方案、需要外援的节点，可以丢给我，
   我拆解成任务，匹配给靠谱设计师，质量我来把控。
   
   目前内测期，想找2-3个项目验证流程。
   您最近有没有强排、户型、立面这种可以外包的节点？
   第一单我可以亲自参与，确保交付质量。」

3. 目标：2周内获得3-5个意向客户
```

**渠道2：精准社群渗透**

```
目标社群：
- 甲方群：开发商设计部群、产业地产群
- 乙方群：各大设计院跳槽群、自由设计师群
- 地区群：各地建筑师交流群

执行动作：
1. 潜伏观察1周，了解群规和活跃时段
2. 输出价值建立信任：
   - 分享你的方法论（wiki内容脱敏后）
   - 回答专业问题（展示专业能力）
   - 分享行业洞察（AI工具、政策解读）
3. 软植入：
   - "最近帮一个朋友对接了立面设计的需求，发现这种模式挺高效..."
   - "整理了一个户型设计 checklist，需要的可以私我"
4. 私信转化：对互动过的人私信介绍平台
```

**渠道3：内容引流（长尾积累）**

```
内容矩阵：

平台          内容形式           频率      目标
─────────────────────────────────────────────────────────
公众号        方法论文章          1篇/周    建立专业信任
知乎          行业洞察+软广       2篇/周    搜索流量

即刻/豆瓣     行业内幕+吐槽       随感      同行共鸣

内容主题示例：

关键：每篇文章底部放平台介绍和联系方式

#### Step 2: 种子设计师招募（第2-3周）

**招募标准（初期宁缺毋滥）**

```
硬性要求：
□ 建筑学专业本科及以上
□ 3年以上工作经验（有完整项目经历）
□ 熟练使用CAD+SU/Rhino+PS
□ 有AI工具使用经验（加分项）

软性筛选：
□ 沟通能力：能否清晰表达设计思路
□ 责任心：过往项目是否按期交付
□ 学习能力：是否愿意接受你的方法论

招募渠道：
1. 你认识的设计师（优先）
2. 设计院离职/转行群体（脉脉、小红书搜"建筑转行"）
3. 自由设计师社群（猪八戒、站酷、小红书）
4. 建筑院校毕业生（导师推荐、校友群）
```

**招募话术**

```
「我在做一个建筑设计任务分发平台RMO-AI，简单说就是：
- 我给你对接设计任务（强排、户型、立面、效果图等）
- 你按任务完成，按劳取酬
- 我负责需求澄清、质量把控、客户沟通

对你的要求：
- 能独立完成任务，质量过关
- 响应及时，每周至少可投入20小时
- 接受我的方法论和质量标准

我能提供的：
- 稳定的任务来源（目标：月均2-5单/人）
- 高于市场的单价（因为质量要求高）
- 你的作品有机会落地建成
- 加入核心设计师群，共享资源

现在内测期，计划招5-8个核心设计师，
有兴趣约个时间聊聊？」
```

**试单机制**

```
目的：验证设计师真实能力，同时让设计师了解平台要求

试单设计：
- 任务：一个小型真实任务或模拟任务
  （如：一个实际项目的立面优化，或一个虚拟项目的强排）
- 报酬：正常结算或象征性补贴（如200-500元）
- 周期：3-5天
- 评估维度：
  1. 设计质量（40%）：专业度、创新性
  2. 交付准时（30%）：是否按时提交
  3. 沟通配合（20%）：需求理解、反馈响应
  4. 文档规范（10%）：文件命名、图层整理

结果：
- 通过：正式签约，开放接单权限
- 待定：给改进建议，再给一次机会
- 不通过：礼貌拒绝，保持联系
```

#### Step 3: 第一单执行（第3-4周）

**执行 checklist**

```
□ 需求确认（客户侧）
  - 约30分钟视频会议
  - 用「需求解析Prompt」输出结构化需求文档
  - 客户确认需求文档，明确交付标准
  - 签署简单服务协议（平台方与客户）

□ 任务拆解
  - 根据方法论拆解任务包
  - 确定每个任务的交付物、标准、工期
  - 生成报价单（客户价格 = 设计师成本 × 1.4-1.6）

□ 设计师匹配
  - 在设计师库中筛选匹配人选（Top 3）
  - 发送任务描述，确认意向和档期
  - 签署设计师合作协议（平台方与设计师）

□ 过程管理
  - 建群：客户+你+设计师（必要时）
  - 关键节点同步（30%/60%/90%进度）
  - 你负责中间层沟通，减少客户和设计师直接摩擦

□ 质量把控
  - 用方法论 checklist 审核中间成果
  - 不符合标准打回修改
  - 必要时你亲自上手调整

□ 交付验收
  - 组织客户验收会议
  - 收集反馈意见
  - 确认修改范围（在合同约定内 or 额外付费）

□ 结算
  - 客户验收通过后3日内结算设计师
  - 你收取平台服务费
  - 收集双方反馈
```

第1单完成标准：
- 客户满意（愿意推荐或复购）
- 设计师满意（愿意继续接单）

- 沉淀了SOP文档（即使有瑕疵）

### 2.1.2 验证指标体系

```
核心指标（北极星指标）：
- 月交易额（GTV）：目标第3个月达到5-10万

一级指标：
├─ 供给侧
│  ├─ 注册设计师数
│  ├─ 通过试单设计师数
│  └─ 活跃设计师数（月接单≥1）
├─ 需求侧
│  ├─ 注册用户/客户数
│  ├─ 发布任务数
│  └─ 成交任务数
└─ 平台效率
   ├─ 任务匹配时长（发布到接单）
   ├─ 平均交付周期
   └─ 纠纷率/退款率

验证标准（第3个月）：
□ GTV ≥ 5万元
□ 活跃设计师 ≥ 5人
□ 成交任务 ≥ 10单
□ 客户复购率 ≥ 30%
□ 设计师留存率 ≥ 60%

如果未达到：
→ 诊断瓶颈（获客？转化？交付？）
→ 调整策略或 pivot
```

### 2.2.2 客户增长策略

**策略1：标杆案例打造**

```
目标：打造3-5个可对外宣传的成功案例

选择标准：
□ 项目有代表性（高容积率/极寒地区/四代宅等）
□ 客户配合度高（愿意提供推荐语、参与案例包装）
□ 交付质量高（有视觉冲击力，可脱敏展示）

案例包装内容：
1. 项目背景卡片
   - 项目区位、规模、难点
   - 客户需求描述

2. 解决方案
   - 任务拆解逻辑
   - AI工具应用点
   - 设计师协作模式

3. 成果展示
   - 前后对比图
   - 关键数据（设计周期缩短X%，成本控制X%）

4. 客户证言
   - 视频采访（最佳）
   - 文字推荐语
   - 后续复购记录

案例应用场景：
- 销售手册/官网展示
- 新客户沟通时的信任建立
- 内容营销素材
- PR稿/行业媒体报道
```

### 2.2.4 平台效率优化

**优化1：智能匹配算法迭代**

```
V1（验证期）：规则匹配
- 硬性过滤：能力标签匹配 + 负载<80%
- 人工排序：你根据经验推荐Top 3

V2（跑通期）：数据驱动匹配
- 引入历史数据：过往接单表现、客户评分、准时率
- 算法排序：匹配分数 = 能力×0.3 + 评分×0.25 + 准时×0.2 + 价格×0.15 + 默契×0.1

V3（规模化）：机器学习
- 训练匹配模型，预测最优接单设计师
- A/B测试验证算法效果
```

**优化2：质检流程自动化**

```
当前流程（人工）：
设计师提交 → 你审核 → 反馈修改 → 你再审 → 交付

优化方向：
1. AI预审（自动）
   - 用RAG知识库检查常见错误
   - 自动检查图层规范、命名规则
   - 输出预审报告，减少人工审核时间50%

2. 分级质检
   - P0任务：你亲自审核
   - P1任务：AI预审 + 抽查
   - P2任务：AI预审 + 客户直接验收

3. 客户自助验收
   - 提供验收checklist工具
   - 关键节点客户在线确认
   - 减少中间沟通成本
```

**优化3：知识库价值释放**

```
知识库应用场景：

### 2.2.5 财务与法务合规

**定价策略**

```
定价模型：

客户报价 = 设计师成本 × (1 + 平台服务费率)

平台服务费率：
- 小单（<5000元）：40-50%
- 中单（5000-20000元）：30-40%
- 大单（>20000元）：20-30%

设计师成本参考：
任务类型        难度    参考报价（元）   工期
────────────────────────────────────────────────────
强排方案        P0      3000-8000       3-5天
户配策略        P1      2000-5000       2-3天
立面概念        P0/P1   4000-12000      3-7天
效果图          P1/P2   1000-3000/张    1-3天/张
景观概念        P1      3000-6000       3-5天

价格调整机制：
- 根据设计师等级上浮/下浮10-20%
- 紧急任务加急费30%
- 长期合作客户折扣10-15%
```

**合同与法务**

```
必备合同模板：
1. 平台服务协议（客户-平台）
2. 设计师合作协议（平台-设计师）
3. 保密协议（NDA）
4. 知识产权转让/许可协议

关键条款：
□ 知识产权归属（通常约定客户拥有，平台/设计师保留署名权）
□ 保密义务（项目信息、客户资料）
□ 质量保证（修改次数、退款条件）
□ 争议解决（仲裁条款）

建议：初期请律师审核合同模板（预算3000-5000元）
```

**财税处理**

```
业务模式选择：

方案A：平台撮合模式（推荐）
- 你作为中介，设计师以个人身份接单
- 客户付款给设计师，你收取服务费
- 设计师自行申报个税，你申报服务费收入
- 风险：设计师完税不可控

方案B：服务外包模式
- 你与设计师签订劳务/外包合同
- 客户付款给你，你再付款给设计师
- 你承担代扣代缴义务
- 成本较高但合规

建议：
- 初期（月交易额<10万）：方案A，简化流程
- 成长期（月交易额>10万）：转为方案B或成立公司
- 咨询专业会计，确保合规
```
