# 对话摘要转生图 Prompt 模板

在调用脚本前，先把对话内容整理成这类摘要。

## 摘要字段

- 目标：
- 主体：
- 动作或姿态：
- 场景或背景：
- 构图或景别：
- 风格或质感：
- 光线：
- 色彩或配色：
- 材质：
- 镜头或视角：
- 画幅或尺寸：
- 画面文字：
- 约束：
- 避免：

## 最终 Prompt 模式

`目标是{目标}，主体为{主体}，{动作或姿态}，场景为{场景}，构图与景别为{构图}，整体风格为{风格}，光线表现为{光线}，色彩与配色为{色彩}，材质细节为{材质}，镜头或视角为{镜头}，画面内文字为{文字}，必须满足{约束}，避免{避免}`

## 改图模式

`保持{必须保留的元素}不变，只修改{本次变化点}，并继续满足{约束条件}，避免{不希望出现的问题}`
